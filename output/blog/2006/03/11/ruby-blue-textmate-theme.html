Ruby Blue Textmate Theme

When did the initial design for the Ruby-Lang Web site a year ago I created a Vim theme that mirrored the colors used in the little code window of the design. Times have changed and I am now an avid Mac user. With the change in operating system I’ve also picked up another text editor called Textmate. Today I decided to spend a little time on a theme for Textmate that mirrored the Ruby Lang design in the same way that my Vim theme did.

To install:

  1. Download the Ruby Blue theme file
  2. Copy it into: /Applications/TextMate.app/Contents/SharedSupport/Themes/

Enjoy!
